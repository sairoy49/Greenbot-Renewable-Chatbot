# 🌱 GreenBot – Renewable Energy Chatbot

GreenBot is a simple, educational chatbot designed to raise awareness about renewable energy sources and home-based energy-saving practices. It helps users learn through a conversational interface that's easy to use and deploy.

## 🚀 Live Demo

[Deploy on Netlify](https://app.netlify.com/drop)  
(*You can drag and drop the `index.html` file directly!*)

---

## 📌 Features

- 💬 Ask about Solar, Wind, and Hydro energy
- 🌍 Get energy-saving tips for everyday life
- 🧠 Built with pure HTML, CSS & JavaScript (no backend)
- 🟢 Deployable on Netlify, GitHub Pages, or any static site host

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- [Netlify](https://www.netlify.com/) for deployment
